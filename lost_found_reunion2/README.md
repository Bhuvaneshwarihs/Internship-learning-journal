# Lost & Found Reunion

An AI-powered system that helps match lost items with found items using image similarity.

## Features
- Upload an image of a lost item
- Search visually similar items
- Image similarity using CLIP
- Fast retrieval using vector database
- Streamlit interface

## Technologies
- Python
- Streamlit
- OpenCLIP
- ChromaDB
- PyTorch

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
